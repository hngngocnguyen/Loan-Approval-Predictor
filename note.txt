    - **Precision** : Parmi les prêts approuvés, % vraiment solvables.
    - **Recall** : Parmi les prêts solvables, % correctement détectés.
    - **F1-Score** : Moyenne harmonique de Precision et Recall.
    - **AUC** : Aire sous la courbe ROC (0.5 = aléatoire, 1.0 = parfait).
    """)

3. Design, Accessibilité & Responsive
Design Responsive (Adapter aux petits écrans)
Utiliser use_container_width=True pour tous les graphiques.

Colonnes adaptatives : Streamlit empile automatiquement les colonnes sur mobile.

Astuce : Testez avec Ctrl+Shift+M dans Chrome.

Accessibilité (Rappels WCAG)
Contraste : Texte lisible sur fond.

Labels clairs : st.number_input("Revenu mensuel net en euros") ✅ vs st.number_input("Rev.") ❌.

Messages explicites : Ne donnez pas juste un code d'erreur technique ; expliquez le problème à l'utilisateur.
